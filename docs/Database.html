Initializes a new Database instance

Example
const Endb = require('endb');
const db = new Endb.Database({
  name: 'endb',
  fileName: 'endb',
  path: './',
  fileMustExist: false,
  timeout: 5000,
  wal: true
});

Properties
Name Type Attributes Description
name string <optional>

The name of the database. Represents the table name in SQLite
fileName string <optional>

The name of the database file
path string <optional>

The path of the database file
fileMustExist boolean <optional>

Whether or not, the database file must exist
timeout boolean <optional>

The number of milliseconds to wait while executing queries on a locked database, before throwing a SQLITE_BUSY error
wal number <optional>

The default method by which SQLite implements atomic commit and rollback is a rollback journal
